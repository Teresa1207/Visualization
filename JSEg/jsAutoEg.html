Illustraing JavaScript Code in HTML

We create a simple HTML document that displays two images. The viewer can change the image on the left using a pull-down menu. She can hide and display the image on the right by toggling the checkbox.

This is very similar to the document jsEg.html. However, here we programmatically create the contents of the SELECT menu, i.e., the OPTION elements. We put information about each animal and its image into an "associative array" or named-element container in JavaScript. We have done this in an unintuitive manner with the text displayed in the optin as the key and then the information about the image and the text to show describing that animal. We should probably have used the file name/URL for the image as the key.

The JavaScript code is stored in a separate file named Eg.js

The CSS file for this HTML document is in eg.css.
