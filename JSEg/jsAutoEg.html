Illustraing JavaScript Code in HTML

This is very similar to the display in jsEg.html, but there are important differences.

  • We create the content of the pull-down menu programmatically.
  • When the viewer selects an animal, we display HTML text/content above the image.
  • We illustrate the alert() JavaScript function.

We put information about each animal and its image into an "associative array" or named-element container in JavaScript. We have done this in an unintuitive manner with the text displayed in the optin as the key and then the information about the image and the text to show describing that animal. We should probably have used the file name/URL for the image as the key.

When the animal changes, there is a relayout that resizes the two images and the layout of the page. This is a little jarring as the user interface controls move up or down. Similarly, this happens when we hide/show the right image. How can we fix this?

  1. We can make all the animal images the same size. We can do this manually outside of the Web page. Alternatively, we can set the width and height attributes of the IMG element in the HTML document.
  2. We can use the visibility style with values visible and hidden. The latter causes the object to take up space but not be shown.

The JavaScript code is stored in a separate file named Eg.js
